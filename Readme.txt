EmbeddedWizard STM32界面开发环境

1.安装 jre-8u261-windows-x64.exe 如果有java环境则不需要安装，具体可以再第二步时候得到提示

2.安装 D:\STM32H747-InterfaceDev\en.stm32cubeprog_v2-5-0\SetupSTM32CubeProgrammer-2.5.0.exe
  获得STM32程序烧写工具

3.安装EmbeddedWizardFree.msi 获取免费版EmbeddedWizard用以开发嵌入式界面

4.安装STM_STM32_PlatformPackage.msi 获取STM32H747的编译环境
